# Calen v1.1.5

> 本版本修复 Skills 商店翻译异常，加入受许可门禁保护的 HY‑MT1.5 离线翻译模型，并将长对话消息导航移到聊天区左侧。

## 翻译修复

- 修复未指定源语言时，Qwen 提示词中的“自动识别”被模型误当成译文返回的问题。
- 本地或远程模型若只返回“自动识别 / auto detect”等占位内容，Calen 会拒绝该结果；本地优先模式可安全回退远程模型。
- 翻译缓存提示版本已更新，旧的异常结果不会继续命中。

## HY‑MT1.5 离线模型

- 默认推荐 `HY-MT1.5 1.8B Q4_K_M`（约 1.13 GB），并提供 `Q8_0` 高质量选项（约 1.91 GB）。
- 下载固定到已核验的 Hugging Face 与 ModelScope revision，并校验精确大小和 SHA-256；安装包本身不包含模型权重。
- 下载或启用前必须明确接受 Tencent HY Community License Agreement、Acceptable Use Policy 与地域限制；欧盟、英国和韩国不在该许可 Territory 内。
- 许可确认以本地 receipt 绑定 model ID 和固定 revision；确认缺失、损坏或过期时，内置 HY‑MT 不会运行。
- Calen 的发布主体为 Tioms。Tencent 与 Tioms/Calen 无关联、无赞助或背书关系。
- Qwen3-0.6B Q8_0 继续作为兼容现有安装及 HY‑MT 许可不适用地区的 Apache-2.0 兜底模型。

## 界面与发布完整性

- 长对话消息导航轨道从聊天区右侧迁移到左侧，悬浮列表向聊天内容方向展开。
- HY‑MT 许可确认框展示固定 revision 和 SHA-256，并支持键盘焦点约束、焦点恢复、失败提示和可复制来源链接。
- Windows 发布检查会强制验证 Tencent HY 许可原文与法定 Notice 已进入安装资源。
