# Calen v1.1.2

> 本版本重点修复 Windows 安装版股票研究服务无法启动的问题，并完善安装器中文体验与股票数据可信度。

## 重点更新

- 修复安装版在 Windows 规范化资源路径（`\\?\C:\...`）下启动股票 sidecar 时出现 `EISDIR` 或 `lstat 'D:'` 的问题。
- 股票服务改为单帧发送 JSON-RPC 请求，传输故障时可自动重启一次并重试只读请求；诊断信息现在保留故障阶段、退出码、时间和脱敏后的 stderr。
- 数据源状态、研究结果和市场结果更严格地区分 `ready`、`partial`、`unavailable` 与“尚未探测”，缺失数据不会被补造成完整结论。

## 股票研究

- 强化 Provider 回退、请求节流、403/429/5xx 冷却、熔断、取消和整单超时处理。
- 改进 A 股、港股、美股与 ETF 的代码解析、来源时效、公司资料和部分失败提示。
- 回测和实验分析补充样本覆盖、费用、基准、算法版本与限制说明，并继续明确标记为实验性研究功能。
- 研究结果按 Provider 和能力展示来源及截至时间；AI 深度研究只基于现有证据，不生成缺失事实。

## Windows

- NSIS 与 MSI 安装界面使用简体中文，并更新安装器、卸载器、应用程序和桌面快捷方式所使用的 Calen 图标资源。
- 安装包继续内置 Node 24 x64 和股票 sidecar，用户无需单独安装 Node。
- 发布前会在中文和空格路径中验证新装、旧版升级、股票 sidecar 启动及卸载清理。

## 说明

- 首版股票功能保持只读，不连接真实券商，也不提供自动交易或保证收益。
- 新浪、BaoStock 及需要 API Key 的 Provider 默认关闭，启用前仍需遵守各数据源条款。
