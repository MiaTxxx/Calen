# Calen v1.1.3

> 本版本修复股票中文显示和 K 线周期切换，并改进聊天历史搜索、预览与系统设置体验。

## 股票研究

- 修复腾讯行情和外汇接口返回 GBK/GB18030 文本时的中文乱码，标的名称不再显示为替换字符。
- 股票快照支持分时、日 K、周 K 和月 K 周期切换；切换只刷新图表数据，不重置整张研究卡片。
- 腾讯日 K、周 K、月 K 使用前复权数据，分时使用分钟接口并还原每分钟成交量。
- 东方财富回退源同步支持四种周期，并统一分钟数据的 ISO 时间格式。
- 分时数据按折线图展示，日 K、周 K、月 K 保持红涨绿跌蜡烛图展示。
- 周期字段进入请求契约、缓存键和响应证据，避免不同周期复用错误缓存。

## 聊天与设置

- 增加聊天历史搜索结果、悬停预览和相对时间展示，长历史记录更容易定位。
- 系统设置补充相关配置项、持久化和同步处理，保持桌面端状态一致。
- 更新股票研究契约测试，覆盖乱码解码、周期路由、缓存隔离和图表数据映射。

## 发布说明

- 本版本继续只发布 Windows x64 的 `Setup.exe`、`.msi` 及对应 updater 签名文件。
- 股票研究保持只读，不连接真实券商，也不提供自动交易或保证收益。
