# Calen 桌面工作区统一语言

股票领域术语见 [UBIQUITOUS_LANGUAGE.md](UBIQUITOUS_LANGUAGE.md)。本表覆盖桌面 GUI 的工作区与扩展能力概念。

| 术语                          | 统一含义                                                                               | 避免使用                           |
| ----------------------------- | -------------------------------------------------------------------------------------- | ---------------------------------- |
| 工作空间（Workspace Project） | 侧边栏里的一个项目条目：名称 + 本地文件夹路径，聚合该目录下的历史对话与 Terminal       | 与「工作目录」「文件夹」混用       |
| Default Project               | 应用管理的兜底工作空间，新对话未指定目录时落在这里；不可重命名、不可移除，可从列表隐藏 | 把它当成普通工作空间并期望删除     |
| 移除工作空间                  | 从列表移除并删除其下历史对话；**不删除**磁盘上的文件夹                                 | 「删除文件夹」                     |
| 隐藏 Default Project          | 仅从侧边栏列表移除显示；兜底目录职责不变，可在设置中恢复                               | 与「移除」混用                     |
| 技能商店（Skills Store）      | Skills Hub 里由 ClawHub 提供的可安装技能列表（`store` 视图）                           | 与「已安装技能」「导入」视图混用   |
| 翻译模型（Translation model） | 设置里为描述翻译单独指定的模型；未设置时回退到当前对话模型                             | 与「标题生成模型」「对话模型」混用 |
